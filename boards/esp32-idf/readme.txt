Twitter: https://twitter.com/airmeterio

This zip file contains a binary build of AirMeter.io for the ESP32 platform. Before using either the shell or batch script ensure that you have:
1) Python 3.4 or newer in the PATH variables. On Windows this can be achieved by installing Python from the Windows Store.
2) Ensure esptool is installed. Simply type pip install esptool. See https://github.com/espressif/esptool

Linux
-----

sh flash.sh /dev/ttyUSB1


Windows
-------
flash.bat COM3
